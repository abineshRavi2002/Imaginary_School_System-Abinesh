# Attendance System

## Setup Instructions

### Server
1. Navigate to the `server` folder.
2. Run `mvn spring-boot:run` to start the server.

### Client
1. Navigate to the `client` folder.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the client.
